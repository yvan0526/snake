# Snake
CLI snake game made with rust.
The game will adapt to the terminal size.
You win the game when no apple is able to spawn.

## Usage
Show help : -h, --help  
Change difficulty : -d \<DIFFICULTY>, --difficulty \<DIFFICULTY> [default: 1, min: 1, max: 20]

## Commands
Control the snake with arrow keys.  
Quit the game with Esc key.
